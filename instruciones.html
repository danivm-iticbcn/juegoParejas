Instruciones del juego de las parejas

- Este es un juego de memoria en el que tienes que encontrar las parejas de las cartas - Haz clic sobre una carta para revelarla, despues clica otra - Si las dos coinciden se mantendran reveladas - Si no coinciden se giraran de nuevo - El juego termina cuando hayas encontrado todas las parejas - Se obtienen puntos por cada pareja encontrada
